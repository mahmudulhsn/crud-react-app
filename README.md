## Step: 1
Clone the repository
```sh 
git clone git@github.com:mahmudulhsn/crud-react-app.git
```
## Step: 2
Go to the project directory
```sh
cd crunch-react-app
```
## Step: 3
Run the command to create the .env file
```sh 
    cp .env.example .env
```
## Step: 4
Run the command
```sh 
    make setup
```

Now your project is ready to run.
